Alien Invasion - with Pygame

This arcade style game is built with pygame modules to emulate a classic invasion scenario. I built several varients including horizontal play, and 3 axis movement to allow the player to approach the invaders. I followed the development instructions from the Python Crash Course written by Eric Matthes,available at no starch press. The book contains several other projects which have been some of the best at promoting learning and new thoughts about how, why and what to program.
